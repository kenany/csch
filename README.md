# csch

Calculate the hyperbolic cosecant of a number.

## Example

``` javascript
const csch = require('csch');

csch(Math.PI);
// => 0.08658953753004692
```

## Installation

``` bash
$ npm install csch
```

## API

``` javascript
const csch = require('csch');
```

### `csch(x)`

Given a _Number_ `x`, returns the hyperbolic cosecant of `x` as a _Number_.
